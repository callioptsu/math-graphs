> [!NOTE]\
> Template for development of front-end projects.

| *Dependencies* | *Version*
| --- | --- |
| **next** | **14.2.1** |
| **node** | **20.0 or higher** |
| **json-server** | **0.17.4** |
| **dotenv** | **16.4.5** |

### Usage

```bash
git clone git@github.com:callioptsu/front-end-template.git

cd front-end-template

npm install && npm run dev
```
